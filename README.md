# 1.Общие сведения
1.1.Данное приложение позволяет загружать данные на удаленный сервер и удалять их оттуда. Имеется возможность зарегистрироваться и войти. Управление производится через web-интерфейс. Приложение использует среду выполнения Node.js. 
1.2.Проект находится на локальном сервере. 
1.3.Проект является учебным.
